# Boston ICE Enforcement Activity - 2025 Data

Administrative arrest statistics for the ICE Boston Area of Responsibility (AOR), covering Massachusetts and New England.

## Data Files

- `boston_ice_arrests_2025.csv` - Structured data file with arrest statistics
- `slide-1.png` through `slide-4.png` - Presentation slides

## Sources

- U.S. Immigration and Customs Enforcement (ICE) ERO Boston

- Deportation Data Project (UC Berkeley)
- Boston Globe analysis of ICE data
- GBH News reporting

## Notes

- Data represents administrative arrests of individuals unlawfully present in the United States
- Some figures are estimates based on available reporting
- Boston AOR covers Massachusetts, Maine, New Hampshire, Vermont, Rhode Island, and Connecticut
- Data current as of December 2025

## Data Structure (CSV)

| Column | Description |
|--------|-------------|
| category | Type of data (monthly_arrests, operations, country_of_citizenship, summary, geographic_focus) |
| subcategory | Specific item within category |
| value | Numeric or text value |
| unit | Unit of measurement |
| time_period | Time period covered |
| source | Data source |
