# Boston ICE Enforcement Activity Data

Administrative arrest statistics for the ICE Boston Area of Responsibility (AOR), covering Massachusetts, Connecticut, Maine, New Hampshire, Rhode Island, and Vermont.

## Data Files

- `boston_ice_arrests_data.csv` - Comprehensive structured data file (FY2021-2025)
- `slide-1.png` through `slide-4.png` - Presentation slides

---

## Historical Annual Arrests (Boston AOR)

|-------------|---------|-------|

| FY2022 | 6,971 | Post-pandemic increase |

| FY2024 | ~11,000 | Estimate based on dashboard |

**Total FY2021-FY2024:** ~29,643 arrests

## Data Structure (CSV)

| Column | Description |
|--------|-------------|
| category | Data type (annual_arrests, monthly_arrests_2025, operations_2025, country_of_citizenship, summary_stats, geographic_focus, detention_facilities, aor_coverage) |
| subcategory | Specific item within category |
| value | Numeric or text value |
| unit | Unit of measurement |
| time_period | Time period covered |
| source | Data source |
| notes | Additional context |

## Notes

- Data represents administrative arrests of individuals unlawfully present in the United States
- Boston AOR covers six New England states: MA, CT, ME, NH, RI, VT
- Some 2025 figures are estimates based on available reporting
- FY = Fiscal Year (October 1 - September 30)
- Dashboard data current as of December 2024 for historical; 2025 data from news reports and ICE releases
